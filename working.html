System Workflow

AgriShe AR follows a simple workflow that allows users to interact with the application and make better farming decisions through AR technology.

1️⃣ Open Application

The user launches the AgriShe AR mobile application and enters the main dashboard.

2️⃣ Choose Plant

The user selects a crop from the plant library according to their farming needs.

3️⃣ Scan Area

The smartphone camera scans the available farming space or kitchen garden.

4️⃣ Generate Preview

The selected crop appears as a virtual AR model in the real environment.

5️⃣ Analyze Layout

The user checks spacing, position and arrangement before actual planting.

6️⃣ Receive Guidance

The application displays reminders and crop care instructions.

Technical Flow: Camera input → AR processing → Plant visualization → User guidance
