AR Enabled Farming Support

AgriShe AR is an Augmented Reality based farming support system created to help women farmers and home gardeners make better planting decisions. By using a smartphone camera, users can preview plants in real space before actual planting.

Project Overview

This project combines agriculture with mobile technology to solve real farming problems through an interactive AR experience.
